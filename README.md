<h1>Game-Page</h1>
<p>Página de jogos criada com servidor em Node.js, a página possui um jogo inspirado no jogo Gartic, junto de outros três mini jogos e uma página de seleção de personagens</p>
<p></p>Conteudo da plataforma:</p>
<ul>
  <li>Home com opção de música</li>
  <li>Jogo Gartic</li>
  <li>Jogo da forca</li>
  <li>Jogo Troca de Palavras</li>
  <li>Jogo da Velha</li>
  <li>Página de seleção de personagens</li>
  <li>Exibição do seu personagem</li>
</ul>
<p>Obs: Para melhor funcionamento dos jogos:</p>
<ol>
  <li>Baixe o arquivo</li>
  <li>Descompacte o arquivo</li>
  <li>Abra o terminal no diretório/pasta</li>
  <li>Use o comando "<i>node server.js</i>"</li>
  <li>Acesse "<i>https://localhost:3000</i>"</li>
  <li>Para outras pessoas na mesma rede acessarem, acesse o IP da maquina que iníciou o programa + "<i>:3000</i>"</li>
</ol>

<div align="center">
  <h3>Jogo Gartic</h3>
  <div align="left">
    <p>O jogo deve ser acessado por dois ou mais jogadores, onde um jogador será o Desenhista enquanto os outros jogadores tem como objetivo descobrir o que está sendo desenhado, o primeiro a acerta ganha 5 pontos e é passado para outro jogador ser o Desenhista</p>
    <p>O desenhista terá as ferramentas para desenhar e a palavra sorteada na parte superior da tela, os demais jogadores não poderão visualizar essa parte</p>
    <p>O campo esquerdo da tela conta com Cards de cada jogador com o Nome e o Pesonagem escolhido</p>
    <p>O campo central é a área de desenho</p>
    <p>O campo direito deve ser usado para acertar a respota</p>
    <p>Na área superior conta com a opção de música caso o jogador deseje</p>
    <p>Chame seus amigos para jogar com você!</p>
  </div>
